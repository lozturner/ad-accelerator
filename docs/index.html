How It Works

🎯

Detects Ads

Watches for ad markers on YouTube and Twitch — no network interception needed

🔇

Mutes Audio

Instantly silences the ad so you don't hear a thing

⚡

Speeds Up

Cranks playback to 2x, 4x, 8x, or 16x — a 30s ad becomes 2s

⏭️

Auto-Skips

Clicks the Skip button the instant it appears

Install in 60 Seconds

  1. Download or clone the GitHub repo
  2. Open chrome://extensions/ in Chrome, Edge, or Brave
  3. Enable Developer mode (toggle in top right)
  4. Click Load unpacked
  5. Select the extension/ folder from the download
  6. Done — click the ⚡ icon to configure
